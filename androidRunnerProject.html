PerfumeJS Plugin implementation to Android Runner plugin

  • Android Runner repo with the plugin inclusion
  • Experiment report
  • Experiment result repo

Implemented a PerfumeJS plugin in Android Runner which aims to measure several performance metrics for mobile web applications. The main metrics focused in this feature were the following:

  • TotalTime which is the request plus response time.
  • DownloadTime which is the response time to a request.
  • First Pain which is the render time of any visual aspect different to what was on screen before navigation to the web page
  • First Contentful Paint which is the render time of the first bit of content from the DOM.

Additionally, performed an experiment to observe the performance metrics generated by the plugin on n 100 web applications randomly selected for which the above performance metrics are collected.
