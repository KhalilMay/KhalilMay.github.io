Driver Distraction Mitigation System

Software Engineering Capstone Project

  • Project's Report
  • Mobile App Source Code
  • Web App Source code

Developed a driver distraction system that monitors the driver’s behaviour while driving and alerts him in case it detects any distractions. It allows a parent or a friend to also monitor the driver from the web application through a map. The system consists of two applications; Mobile App and Web App.

Driver Disctraction Mobile App

Mobile App developed in Android Studio using Java with data stored in Firebase. The application allowed the user to start a session when he starts driving, and during the session distraction mitigation measurement were made for the user to stop using the mobile device. The below are some main features in the app:

  • Screen touch detection to prevent the driver from using the device during the session.
  • Accelerometer sensor reading during the session to prevent driver from exceeding speed limit.
  • Voice message feature to automatically play a voice message sent from parent account.
  • Device location collected for parental viewing.
Driver Disctraction Web App

Web App developed in Visual Studio using ASP.Net web forms with data stored and retrieved from Firebase. The web application will process data and send the results back to the mobile application. Also it will have a map to track the user’s phone location. Below are some main features in the web app:

  • The web app will process data received through APIs, on the web-based backend.
  • Data of driver's sessions are displayed to the web app user.
  • Web app displays driver's location in a map for parental user to track.
  • Web app allows parental user to send text message to driver which then the mobile app uses Text-to-Speech to play for the driver.
