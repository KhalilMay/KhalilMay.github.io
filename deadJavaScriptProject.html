Impact of Dead Javascript Code Removal

Computer Science Master Thesis Project

  • Experiment's Source Code
  • Report

Used LacunaV2 tool to remove dead JavaScript code from 36 selected web applications, then performed an experiment using AndroidRunner tool to measure energy consumption, memory consumption, loading time, first paint(FP), and first contentful paint(FCP) of the subjects before and after dead JavaScript removal. Furthermore, FP and FCP were collected by using Perfume.js package injected into all the websites in the experiment.

The result of the experiment shows whether removing dead Javascript code improve the performance of the websites. Details of the expreiment can be found in the "Report" linked above.
